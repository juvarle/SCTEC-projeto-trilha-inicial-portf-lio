Portfólio Pessoal

Este projeto consiste no desenvolvimento de uma página de portfólio pessoal com o objetivo de apresentar minhas habilidades, conhecimentos e projetos desenvolvidos ao longo do curso Carreira Tech 2026. A proposta foi criar um site simples, organizado e responsivo, utilizando as principais tecnologias aprendidas.

A página foi estruturada utilizando HTML, garantindo um conteúdo organizado e acessivel. Foram utilizadas tags como header, section, article e footer para dividir corretamente cada parte do site, como a apresentação pessoal, habilidades, projetos e contato.

Na parte de estilização, foi utilizado CSS, aplicando conceitos como reset de estilos, uso do box model, Flexbox para organização dos elementos e media queries para garantir a responsividade em diferentes tamanhos de tela. Também foi definida uma paleta de cores em tons de roxo pastel para deixar o visual mais agradável, moderno e delicado.

Para a interatividade, foi utilizado JavaScript, permitindo a manipulação do DOM. Foi implementado um formulário de contato com validação simples, onde o usuário recebe uma mensagem de confirmação ao enviar seus dados.

A página contém seções como "Sobre mim", "Habilidades", "Projetos" e "Contato". Na parte de projetos, são apresentados atividades desenvolvidas durante o curso, com descrição e tecnologias utilizadas.

Este projeto foi importante para consolidar conhecimentos básicos em desenvolvimento web, além de proporcionar experiência prática na criação de páginas estruturadas, estilizadas e interativas.

Para visualizar o projeto, basta abrir o arquivo index.html em qualquer navegador. Obrigada pela oportunidade e espero que gostem.
